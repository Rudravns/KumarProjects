Chess Project Overview

This is a full chess game library combining Python for frontend visuals and C# for backend logic. Players can enjoy classic chess with move validation and interactive gameplay for two players.

Key Features:

  • Full chess rules with legal move validation
  • Two-player mode on a single device
  • Clean interface using Python graphics libraries
  • Backend logic in C# for fast computation

Development Timeline

Step 1: Planning & Design

Outlined project requirements, designed the chessboard layout, and planned features.

Step 2: Start Frontend designing

Implemented Chess board, added colors, and a debug menu

Step 3: Frontend UI and Datatypes in Python

Created the interactive board and pieces, created many data types for better organization, and added MORE colors.

Step 4: Fixing Bugs and Folder Management

Tested gameplay, fixed bugs, optimized the interface for smooth play, and organized folders for C#.

Step 5: Linking Backend to Frontend

Created a Python code to specifically link python to c#.

Step 6: Created Backend Logic in C# and FEN loading/Saving

Created c# code to get legal moves, check for checkmate, promotion, and more. Also added FEN loading and saving functionality.

Step 7: FEN Loading and Saving and Promotion

Fixed many bugs, fully completed the saving and loading functionality of the game, and created promotion UI.

Step 8: PGN support and Finalizing

Added PGN support for full game readibilty and finishing the project

Step 9: Backtracking and More Bugs

Added Bug fixes and Backtracking support.

Step 10: Publishing

Published on Github for everyone to see and use!
