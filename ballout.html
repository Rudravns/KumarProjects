Ballout

A fast-paced physics-based ball game built with Python and Pygame.

🎮 Play / Download on Itch.io 📦 View Source on GitHub

About the Game

Ballout is a small fast-paced arcade-style physics soccar game where players control a ball and try to score while avoiding other players.

The game focuses on responsive movement mechanics, collision detection, and quick reaction gameplay.

Why was this game created

This game was created for a coding challange and was based on a past soccar game with only controller support (it was very fun). It was made because I wanted a fun game that me and my sister could play together. This current game has bot and a little multiplayer support.

Gameplay

  • Control a moving ball
  • Avoid opponents
  • Maintain momentum and contoll of the ball
  • Score as many goals as possible in 90 seconds

Controls

  • Move (WASD for players on right and arrow keys for players on left): WASD/Arrow Keys
  • Pause: ESC

Project Structure

    Ballout.py       → main game loop
    Ballout_classes.py     → repeated non-game-goal classes/funcs
    Ballout_entities.py      → ball and player classes
    ballout_multiplayer.py    → multiplayer support (not fully implemented)
    README.md         → game discription
    Ballout.exe  → executable game file
    ballout_assets/       → textures, sounds, fonts
