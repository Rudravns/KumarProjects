Raycaster

A classic raycasting renderer built with Python and Pygame.

What is Raycasting?

Raycasting is a rendering technique used by early 3D games to simulate a three-dimensional world from a 2D map. Instead of rendering full 3D geometry, the engine casts rays outward from the player's view to calculate wall distances.

This project recreates that technique using Python and Pygame to better understand how classic game engines worked.

Engine Features

  • Real-time raycasting renderer
  • Grid-based world map
  • Wall collision detection
  • Player movement and camera rotation
  • Distance-based wall scaling

Controls

  • W / Up Arrow — Move Forward
  • S / Down Arrow — Move Backward
  • A — Rotate Left
  • D — Rotate Right
  • ESC — Exit

Technical Details

  • Python + Pygame rendering
  • DDA raycasting algorithm
  • 2D grid world map
  • Efficient collision system

Project Structure

main.py      → main loop
player.py    → player movement
raycaster.py → raycasting engine
map.py       → world map
