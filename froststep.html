Froststep

A top-down survival game built with Pygame.

Created for the Pygame Community Winter Jam 2026

🎮 Play / Download on Itch.io 📦 View Source on GitHub

What is Froststep?

Froststep is a top-down survival game where the player explores a frozen world, gathers resources, and fights to keep a central beacon alive.

The environment is covered in darkness and freezing fog. Your warmth slowly drains as time passes, forcing you to explore the world, harvest resources, and return them to the beacon before the cold overtakes you.

Gameplay Loop

  • Gather wood and membrane from trees.
  • Carry resources in a 3-slot inventory.
  • Return resources to the Beacon to restore warmth.
  • Survive roaming enemies such as spiders.
  • Manage your warmth meter and time left.

The longer you survive and the more fuel you deposit, the stronger the beacon becomes — expanding its light radius and pushing back the darkness.

Controls

  • Move: WASD / Arrow Keys
  • Interact / Pick up / Deposit: E
  • Drop item: Q
  • Switch inventory slot: Mouse wheel
  • Toggle fullscreen: F11

Debug Keys

  • F1 – Toggle UI debug overlay
  • F2 – Toggle hitbox debug
  • F3 – Toggle console debug output

Enemies & AI

The frozen forest is not empty. Spiders wander the world using a simple state-based AI system:

  • Wander: Random roaming around the map
  • Chase: Lock onto the player when detected
  • Attack: Strike when within range

This forces players to balance exploration with survival.

Technical Challenges

Froststep renders a large 5000×5000 world with hundreds of objects while maintaining smooth gameplay.

  • Optimized collision detection using Pygame Rect lists
  • Dynamic fog-of-war using alpha surfaces
  • Resolution scaling for fullscreen and windowed modes
  • Efficient sprite and asset loading utilities

Project Structure

        main.py       → main game loop
        player.py     → player logic
        enemy.py      → spider AI
        objects.py    → trees and world objects
        ui.py         → UI system
        utilities.py  → helper utilities
        Assets/       → textures, sounds, fonts
