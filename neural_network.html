Neural Network Basics

A fully-implemented neural network from scratch that recognizes handwritten digits (0-9) from the MNIST dataset, with an interactive Pygame visualization interface.

Key Features:

  • Custom Neural Network implementation using NumPy
  • Multi-layer perceptron with ReLU hidden layers and Softmax output
  • Interactive Pygame interface for drawing digits
  • MNIST dataset support (training & testing)
  • Model persistence with JSON save/load

Project Structure

Machine-Learning/
├── Main.py        # Pygame interface
├── network.py     # Neural network implementation
├── load_data.py   # MNIST loader
├── helper.py      # Helper functions and save/load
├── model_data.json
└── data/          # MNIST dataset

Installation & Usage

  1. Install dependencies: pip install numpy scipy pygame matplotlib
  2. Download MNIST dataset into data/
  3. Run application: python Main.py

Command-line options:

  • --learning Train a new model from scratch
  • --test_acc Evaluate test accuracy
  • --times N Repeat training N times

It will take aprox. 6-7 mins to train this Model
