What is Minecraft?

A 3D sandbox video game focused on creativity and exploration. Players explore a procedurally generated world made of blocks, gathering resources to build structures in a "sandbox" with no set goals.

Controls

  • WASD to move
  • Right click to Build
  • Left click to Destroy
  • E to open inventory
  • Left joystick to Move

    • ------------ Dualsense Controller ----------------

  • Right joystick to See
  • R2 to Destroy
  • L2 to place
  • X to jump
  • r1 to go to next inventory slot
  • l1 to go to previous inventory slot

Technical Details

Minecraft was built using Python and the pygame modified Opengl framework. The game features procedural generation, building, crafting, destruction, and responsive movement controls wih dualsense controller controls.

  • Python
  • Pygame
  • Open-Gl
