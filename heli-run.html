What is HELI RUN?

HELI RUN is a fast-paced arcade helicopter game where the player must fly through an endless set of obstacles while maintaining control of the helicopter.

The goal is to survive as long as possible while navigating through tight spaces and increasingly difficult terrain.

Gameplay

  • Fly a helicopter through side-scrolling obstacles
  • Avoid crashing into pipes and terrain
  • Survive as long as possible
  • Difficulty increases as the game progresses

Controls

  • SPACE / Mouse Click: Fly Up
  • Release: Descend
  • ESC: Pause / Exit Menu

Technical Details

HELI RUN was built using Python and the Pygame framework. The game features procedural obstacle generation, dynamic difficulty scaling, and responsive movement controls.

  • Python
  • Pygame
  • Dynamic obstacle generation
  • Collision detection system
